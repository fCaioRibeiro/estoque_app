# Installation

1. Clone the repo

```sh
git clone https://github.com/fCaioRibeiro/estoque_app.git
```

2. Install the packages using NPM

```sh
npm install
```
3. run the project

```sh
npm start
```

4. Scan the QR code above with Expo Go (Android) or the Camera app (iOS).

Happy coding!
